# Backup Automation Script

This project is a simple Bash script that automates file backups.
It copies files from a source directory to a backup directory and logs all operations.

## Project Structure

backup_project/
├── backup/
├── logs/
├── origem/
├── backup.sh

---

## How to Run

```bash
chmod +x backup.sh

## What This Script Does

Creates a backup directory if it does not exist

Copies files from the source folder

Generates a log file with date and time

Confirms if the backup was successful
